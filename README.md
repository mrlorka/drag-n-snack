# drag-n-snack

## Project setup
```
npm install
```

### Setup Mock DB
```
npm run mock
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
